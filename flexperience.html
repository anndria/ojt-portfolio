FLEXPERIENCE

This page presents the case study of Flexperience, an online fitness facility locator in Davao City.
It includes an overview of the project, the tools utilized, and a link to the project repository on GitHub

PROJECT LINK

Project Overview

Flexperience is a web-based platform designed to address the challenges associated with locating fitness centers and related services in Davao City. Targeted primarily at fitness enthusiasts, the platform provides a comprehensive solution for finding gyms, personal trainers, and tailored workout and meal plans. Through a user-friendly and centralized interface, it simplifies the process of discovering fitness options, allowing users to navigate various offerings with ease.

Leveraging advanced mapping technology and user-generated reviews, Flexperience enables users to find new workout locations, whether they prefer gym environments or outdoor sessions. The platform integrates a wide range of fitness gyms across Davao City, offering registered users seamless access to facilities and the ability to book personal trainers. This feature is designed to provide professional fitness guidance and convenient access to resources, ultimately supporting users in achieving their fitness goals.

Tools Utilized

Figma

We use Figma to design our interface for both PC and mobile responsiveness, allowing us to create seamless, adaptive layouts that ensure a consistent user experience across different devices.

HTML

We use traditional HTML for this project because it is a safe and reliable choice for our group, ensuring stability and compatibility across platforms.

Tailwind CSS

We use Tailwind CSS for this project because it provides a highly customizable, utility-first framework that allows for rapid and efficient styling.

Express.js

We use Express.js for this project's backend because it is a flexible and minimalistic framework that allows for the rapid development of web applications.

JavaScript

We use JavaScript for the responsiveness of the front-end because it enables dynamic, interactive elements and ensures a smooth, adaptive user experience across different devices.

MySQL

We use MySQL for this project’s database because it is a proven, efficient relational database management system that provides reliable data storage and management capabilities.
